== Description ==

Make Responsive Mobile Friendly Price/Pricing Table for Any where of your Website. This plugin will provide a shortcode like [] and User able to show Price table anywhere. Easily show a Price table at Any where of your website, just using a shortcode. No need Programming knowledge. Available different template.

**[Demo Price Table](https://wooproducttable.com/purchase-table/)**

**Easy Price Table** plugin offers to display your Price Table in page by shortcode as table. 

* Easy Customize able
* Unlimited Table
* Unlimited Element

**Elements of Price Table**

* Content (WP Editor)
* Price
* Divider
* Spcacer
* Heading Text/name
* Link/ Buy Button/ Subscribe Button
* Banner Image

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/easy-price-table` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Dashboard -> Easy Price Table -> Add new.


== Frequently Asked Questions ==

= Is it Responsive =

Yes.

= What is default Shortcode? =

Default Shortcode is [EASY_PRICE_TABLE id='123'] . Here 123 is your easy_price_table 's POST ID.

= How to use? =

Install and activate. Then go to ( Dashboard -> Easy Price Table -> Add New ).
